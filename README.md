# Akram Sweets Web App

A modern and responsive eCommerce sweets store web application built using React.

## Features

- Home page with featured products
- Product categories
- Shopping cart functionality
- Responsive design for all devices
- User-friendly interface
- Fast and smooth navigation

## Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3

## Installation

Clone the repository:

bash
git clone https://github.com/Tiirow/Akram-sweets-wep-app.git


Install dependencies:

bash
npm install


Run the application:

bash
npm start
